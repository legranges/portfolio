1.Static website

A static website is a type of website where the content is pre-built and served to users exactly as stored, without any server-side processing or interaction with databases. These websites are typically created using HTML, CSS, and optionally JavaScript for client-side interactivity. The content remains consistent for all users and only changes when a developer manually updates the source code.



2.Dynamic website

A dynamic website is a site that generates pages in real time, responding to dynamic characteristics such as screen size and device type. The structure and content of a dynamic web page are flexible, allowing you to customize the end-user’s experience based on the browser or requests.
